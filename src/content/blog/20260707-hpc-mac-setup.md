剛開始準備進行 DFT 與 VASP 計算時，第一個面臨的挑戰往往不是複雜的量子化學公式，而是如何搞定與超級電腦的連線！由於許多實驗室慣用的 MobaXterm 只有 Windows 版本，Mac 使用者通常需要尋找替代方案。

這篇日誌紀錄了我在 Mac 上使用 Termius 連線至國網中心（Taiwania 3）時踩過的坑，以及後續的 Linux 終端機設定與檔案管理解法，希望能幫到同樣在這條路上卡關的  mac用戶。

---

## 🚧 關卡一：Termius 連線失敗與 2FA 無限輪迴

**遇到什麼問題？**
在使用 Termius 設定好 SSH 連線後，一直卡在紅色的連線失敗畫面；有時候即使跳出了請選擇 2FA（雙重認證）的選項，輸入數字後卻無限輪迴，遲遲等不到 6 位數驗證碼。

**解決辦法與真相：**
這其實不全是 Termius 軟體的 Bug，而是國網中心的帳號設定問題被圖形化介面隱藏了。

1. **用 Mac 原生終端機抓漏：** 直接打開 Mac 內建的 Terminal，輸入 `ssh 你的帳號@twnia3.nchc.org.tw`。這時終端機誠實地顯示了隱藏的錯誤訊息：`[FAIL] Please complete iService OTP registration to authenticate.`
2. **開通 OTP 服務：** 原來是因為還沒在國網中心後台綁定雙重認證！前往「國網中心 iService 平台」，登入後在會員中心找到「OTP 驗證設定」，強烈建議綁定 Mobile APP OTP（如 Google Authenticator）。
3. 設定完成後，回到 Termius 重新連線就能順利通過了！

## 🚧 關卡二：用 Vim 編輯 .bashrc 設定快捷鍵卡住

**遇到什麼問題？**
為了解省打指令的時間，指導老師要求在 `.bashrc` 檔案中新增快捷指令（alias）。但在 Vim 編輯器裡面不知道怎麼打字，存檔時也一直跑出奇怪的畫面。

**解決辦法與 Vim 基礎操作：**
* **進入編輯模式：** 按下鍵盤的 `i` 鍵，確認左下角出現 `-- INSERT --` 才能開始修改。
* **正確解除註解：** 找到 `alias` 那一行，必須把最前面的 `#` 與空格刪除（在 Linux 中 `#` 代表註解，系統會忽略該行）。
* **新增帳號：** 補上完整的指令，例如 `alias qq='qstat -u u7826331'`。
* **存檔離開（關鍵）：** 按下 `Esc` 退出編輯模式，接著輸入 `:wq`（一定要打冒號），然後按 `Enter` 即可成功跳出。
* **套用設定：** 回到終端機後，務必輸入 `source ~/.bashrc`，剛剛設定的 `qq` 捷徑才會正式生效。

## 🚧 關卡三：處理帶有空格的中文檔案與搬移

**遇到什麼問題？**
透過 SFTP 將檔案（例如：`413191438 曾智凱 物理實驗二 結報.docx`）上傳後，檔案預設會落在容量極小的「家目錄（`/home`）」，需要將它移動到專屬的「工作目錄（`/work/你的帳號`）」。但檔名太長又有空格，手打指令非常容易報錯。

**解決辦法與 Linux Tab 神技：**
千萬不要自己傻傻手打又長又有空格的檔名，請善用 Linux 的自動補齊功能：

1. 輸入移動指令 `mv` 與檔名的前幾個字：`mv 413`。
2. **按下 Tab 鍵：** 系統會瞬間幫你把整串檔名補齊，而且會自動加上單引號來保護檔案名稱中的空格（變成 `'413191... .docx'`）。
3. 空一格後，接上目的地路徑：`/work/你的帳號/`。
4. 完整指令如：`mv '413191438 曾智凱 物理實驗二 結報.docx' /work/u7826331/`
5. 按下 `Enter` 後，如果系統沒有跳出任何訊息直接換行，這在 Linux 裡就代表「搬移成功」！最後用 `cd` 切換過去並用 `ls` 檢查，檔案就乖乖躺在那裡了。
